### githubpages

1. Znajdź templatke jaka Ci się podoba
2. fork it
3. nazwij repo \<your user name\>.github.io
4. zmodyfikuj \_config.yml
5. dodaj nowy post
   _posts/2024-03-11-my-first-blopost.md

Pliki powinny być w formacie: yyyy-mm-dd-your-blog-post-name.md
